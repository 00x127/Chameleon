Chameleon

Anti-fingerprinting for Firefox. Per-origin deterministic noise; nothing leaves the browser.

Protection
Master switch. Turn off to disable globally without losing your settings.

Browser identity

Auto picks Linux, Windows, or macOS per site so cross-site comparison cant match on OS. Pick a fixed profile if you want every site to see the same one.

Surfaces

Exempt origins

eTLD+1 entries, one per line. Protection is fully disabled for these sites.

What this blocks

  • Canvas 2D readback (toDataURL, toBlob, getImageData), per-pixel LSB noise keyed to origin
  • WebGL / WebGL2 parameter spoofing, readPixels noising, getShaderPrecisionFormat normalisation
  • WebGPU hidden entirely (navigator.gpu returns undefined)
  • AudioBuffer / AnalyserNode readouts (sparse float perturbation, inaudible)
  • TextMetrics, getBoundingClientRect, getClientRects, offsetWidth/Height, scrollWidth/Height (sub-pixel jitter)
  • FontFaceSet.check coerced to a stable common-fonts answer
  • Screen dimensions, devicePixelRatio, outer dimensions, screenX/Y
  • Navigator: userAgent, platform, oscpu, vendor, hardwareConcurrency (8), deviceMemory (8), languages, plugins, mimeTypes, maxTouchPoints, userAgentData
  • NetworkInformation, Battery, Storage estimate, MediaCapabilities, mediaDevices.enumerateDevices
  • SpeechSynthesis.getVoices
  • matchMedia for prefers-color-scheme, prefers-reduced-motion, prefers-contrast, forced-colors, pointer, hover, color-gamut, dynamic-range
  • WebRTC ICE host / srflx candidate suppression (local IP leak); relay candidates kept so video calls work
  • Permissions API normalised
  • Motion, orientation, generic sensor APIs blocked
  • Bluetooth, HID, Serial, USB, Wake Lock, Ink, Clipboard surfaces hidden
  • indexedDB.databases() list, navigator.getGamepads()
  • Worker constructor wraps remote scripts and installs hooks inside the worker scope
  • Web MIDI: requestMIDIAccess rejected, MIDIAccess and related globals hidden
  • Ad-blocker detection bypass: bait elements report non-zero dimensions, common detection globals pre-populated, so paywall nags don't trigger
  • Firefox quirks (InstallTrigger, mozInnerScreenX/Y, netscape, sidebar, CSS.supports moz prefixes) hidden so feature detection cant confirm Firefox
  • Font enumeration locked down: FontFaceSet (size, iteration, check, has, forEach), navigator.fonts.query, window.queryLocalFonts blocked. Canvas measureText returns fallback widths for non-standard fonts so JS-attribute font detection fails. Visual rendering of custom fonts is preserved.
  • Cross-origin Referer stripped, Origin replaced with destination origin on cross-site requests
  • HTTP layer: User-Agent, Accept-Language, Sec-CH-UA family, Accept-CH, NEL/Report-To, Save-Data, Device-Memory
  • Timer precision: performance.now, Date.now, event.timeStamp, requestAnimationFrame callback ts, PerformanceEntry, PerformanceTiming
  • Intl normalisation: NumberFormat, Collator, RelativeTimeFormat, PluralRules, ListFormat, DisplayNames, Segmenter all report en-US / latin / gregory
  • Timezone pinned to UTC (offset only, not the wall clock)
  • PaymentRequest, file pickers, error stack scrubbing
  • Function.prototype.toString camouflage so hooks read as native code
